Event¶

class gem.event.Event(key)¶

Event is an object which Listeners can register to be notified by when certain events occur.

This is the basis of Gem’s event driven API. Many events are provided and triggered by the kernel, however events can also be created and triggered from within python code.

Parameters:key (string) – A unique identifier for this event

Attributes

key¶

The unique identifier for this event

Methods

register(listener)¶

Subscribes a PyListener to this event

Parameters:listener (PyListener) – The listener to register
unregister(listener)¶

Removes a PyListener from the list of listeners for this event

Parameters:listener (PyListener) – The listener to unregister
notify_observers(*args)¶

Notifies all registered listeners that this event was raised, and passes down an arbitrary set of arguments.

Parameters:args (list) – an arbitrary list of arguments to provide to listeners
