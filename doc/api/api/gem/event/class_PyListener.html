PyListener¶

class gem.event.PyListener(callback)¶

A PyListener is an observer of a Event which is tied to a python function or method.

When the listener is notified, callback is called with the list of arguments passed to notify_observers()

Parameters:callback (function) – The function to call when this listener is notified

Attributes

id¶

The unique identifier for this listener
