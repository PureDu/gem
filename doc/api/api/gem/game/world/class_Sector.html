Sector¶

class gem.game.world.Sector¶

Sector is an instance of an 8x8 chunk of the world. It’s primary function is to track which entities are in the sector.

Note

Should not be constructed from Python

Attributes

active¶

Whether this sector is active. Inactive sectors are garbage collected when WorldInstance.gc() is called.
