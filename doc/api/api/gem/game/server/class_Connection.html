Connection¶

class gem.game.server.Connection¶

Connection is a network-level representation of a client. It handles read/write buffering, and decodes data into game packets or update requests for processing.

Note

Should not be constructed from Python

Attributes

log¶

The gem.log.Module associated with this connection

Methods

disconnect()¶

Closes the connection forcibly. Non-blocking.
