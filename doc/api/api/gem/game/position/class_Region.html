Region¶

class gem.game.position.Region(origin)¶

A region is a 13x13 sector (104x104 tile) chunk. This is mainly used to represent the loaded area of the map around the player

Parameters:origin (Sector) – The sector located at the lowest corner of the region

Attributes

origin¶

The Sector located at the lowest corner of the region (ie. NOT the center of the region)

Methods

rebase(position)¶

Rebase adjusts the region such that it;s new center is the sector containing the given position

Parameters:position (Absolute) – The coordinate to rebase onto
