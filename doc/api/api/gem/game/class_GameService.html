GameService¶

class gem.game.GameService(context, rsa_key, auth_provider)¶

GameService is the service handling game clients

Parameters:
  • context (gem.runite.Context) – The runite context to serve content from
  • rsa_key (string) – The path to the RSA keypair to use for secure login
  • auth_provider (gem.auth.ProviderImpl) – The object implementing gem.auth.ProviderImpl to use for player authentication

Attributes

world¶

The gem.game.world.WorldInstance of this game service
