EntitySlice¶

class gem.game.entity.EntitySlice¶

EntitySlice is a slice of entities.

EntitySlices can be added to and emptied, but not removed from. They are intended for buffering entities for addition to a EntityList.

Attributes

size¶

The number of entities in the slice

list¶

A python list containing the entities in the slice.

Methods

add(entity)¶

Adds an entity to the slice

Parameters:entity (Entity) – The entity to add
empty()¶

Empties the slice

filter(type)¶

Creates a new slice, which is the subset of entities with the given type

Parameters:type – The entity type to select
Returns:The new filtered slice
Return type:EntitySlice
