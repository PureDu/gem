ProviderImpl¶

class gem.auth.ProviderImpl¶

Base class for authentication providers

Warning

Should not be used directly. This is a base class whose methods should be implemented by your own python class.

See also

Your auth provider implementation should be passed as a parameter to gem.game.GameService

Methods

load_profile(username, password)¶

Authenticates and loads the player’s profile.

Parameters:
  • username (string) – The player’s username
  • password (string) – The encrypted form of the player’s password
Returns:

Tuple of profile and auth response
Return type:

(gem.game.player.Profile, auth response)
